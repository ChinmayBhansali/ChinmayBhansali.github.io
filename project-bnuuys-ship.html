Bnuuy's Ship

A 2D video game built from scratch with a custom Entity-Component-System (ECS) game engine using C++ and OpenGL.

Role: Gameplay & UI/UX Developer

Tech Stack: C++, OpenGL, ECS

Team: 5 members

▶ Play Demo GitHub

Overview

Bnuuy's Ship is a top-down action/strategy game where you captain a modular ship through a flooded world, rescuing bunnies from scattered islands while battling pirate chickens. Inspired by Lovers in a Dangerous Spacetime, players manage multiple ship stations, steering, cannons, lasers, and healing modules, while assigning rescued bunnies as AI-powered crew members.

Built from scratch using a custom Entity-Component-System (ECS) engine with C++ and OpenGL, the project challenged us to implement everything from rendering pipelines to physics systems without relying on existing game engines. The result is a polished, self-contained experience with intro cutscenes, an in-game encyclopedia, save/load progression, and dynamic audio, playable start-to-finish with no external instructions needed.

Technical Highlights

  • Custom ECS Architecture - Built from scratch for efficient entity management, enabling modular ship components and dynamic crew assignment.
  • A* Pathfinding - Enemy ships intelligently navigate around island obstacles with dynamic path recalculation.
  • Polygon Collision System - Frame-consistent polygon-polygon collision detection with sliding resolution for smooth ship navigation.
  • Decision Tree AI - Helper bunnies autonomously operate stations using decision tree logic for automated defense.
  • Instanced Particle Rendering - GPU-optimized particle system for visual effects while maintaining stable 60 FPS.
  • Multi-Resolution Support - Responsive window scaling with consistent aspect ratio across different screen sizes.
  • Tiled Map Integration - Tileson library for parsing complex maps with spawn points, collision polygons, and layered rendering.
